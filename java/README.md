# ☕ Java Learning Path — Beginner to Advanced

A comprehensive collection of Java programs organized from beginner to advanced level. Each file is self-contained with detailed comments explaining the concepts.

## 📁 Structure

| Topic | File | Concepts |
|-------|------|----------|

| Control Flow | `Loops.java` | for, while, do-while, for-each, break, continue |
| Arrays | `ArrayBasics.java` | 1D arrays, Arrays utility methods |
| Arrays | `MultiDimensionalArrays.java` | 2D arrays, jagged arrays |

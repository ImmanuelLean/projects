# 🚀 Full-Stack Project Curriculum

Build 5 real-world projects — from beginner to advanced.
Each project builds on skills from the previous one.

## Projects

| # | Project | Level | Key Skills |
|---|---------|-------|------------|
| 1 | **Blog Platform** | Beginner | CRUD, Auth, REST API, React basics |
| 2 | **Task Manager API** | Beginner+ | API design, validation, testing, docs |
| 3 | **E-Commerce Store** | Intermediate | Payments, cart, orders, admin panel |
| 4 | **Real-Time Chat** | Intermediate+ | WebSockets, rooms, online status |
| 5 | **Social Media App** | Advanced | Feeds, followers, notifications, scaling |

## Tech Stack

- **Backend**: Node.js, Express, Prisma ORM
- **Database**: PostgreSQL + Redis
- **Frontend**: React, React Router, Zustand/Redux
- **Auth**: JWT + bcrypt
- **Testing**: Jest + Supertest
- **Deployment**: Docker + CI/CD

## How to Use

1. Read each project's `guide.md` step-by-step
2. Build it yourself — don't copy-paste
3. Add your own features after finishing
4. Deploy each project
5. Add to your GitHub portfolio

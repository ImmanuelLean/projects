# Emmanuel's Coding Cheatsheet 🚀

## Open Projects
```bash
cd ~/projects
code .          # opens VS Code
```

## Run Code
```bash
python3 file.py                    # Python
node file.js                       # JavaScript
g++ file.cpp -o output && ./output # C++
gcc file.c -o output && ./output   # C
```

## Save to GitHub (3 commands)
```bash
cd ~/projects
git add .
git commit -m "describe what you changed"
git push
```

## Create New Folder
```bash
mkdir ~/projects/new-folder-name
```

## Useful Commands
```bash
ls              # list files
cd folder       # go into folder
cd ..           # go back
pwd             # show current location
clear           # clear screen
```
